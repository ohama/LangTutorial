## v1 Requirements

### Foundation (Phase 1)

- [x] **FOUND-01**: .NET 10 + FsLexYacc 프로젝트 구성 설명
- [x] **FOUND-02**: fslex로 토큰 생성 (Lexer 기초)
- [x] **FOUND-03**: fsyacc로 AST 생성 (Parser 기초)
- [x] **FOUND-04**: Discriminated Union으로 AST 타입 정의

### Expressions (Phase 2)

- [x] **EXPR-01**: 사칙연산 (+, -, *, /) 구현
- [x] **EXPR-02**: 연산자 우선순위 처리 (*, /가 +, -보다 먼저)
- [x] **EXPR-03**: 괄호로 우선순위 변경
- [x] **EXPR-04**: 단항 마이너스 (음수) 지원

### Variables (Phase 3)

- [x] **VAR-01**: let 바인딩 (let x = 5)
- [x] **VAR-02**: 식에서 변수 참조
- [x] **VAR-03**: let-in 식으로 지역 스코프

### Control Flow (Phase 4)

- [x] **CTRL-01**: if-then-else 조건 분기
- [x] **CTRL-02**: Boolean 타입 (true, false 리터럴)
- [x] **CTRL-03**: 비교 연산자 (=, <, >, <=, >=, <>)
- [x] **CTRL-04**: 논리 연산자 (&&, ||)

### Functions (Phase 5)

- [x] **FUNC-01**: 함수 정의 (let f x = x + 1)
- [x] **FUNC-02**: 함수 호출 (f 5)
- [x] **FUNC-03**: 재귀 함수 지원
- [x] **FUNC-04**: 클로저 (외부 변수 캡처)

### Quality (Phase 6 - ON HOLD)

- [ ] **QUAL-01**: 사용자 친화적 에러 메시지
- [ ] **QUAL-02**: 대화형 REPL 셸
- [ ] **QUAL-03**: Expecto, FsCheck, fslit으로 테스트

### CLI & Tests (Phase 7)

- [x] **CLI-01**: 입력 방식 — --expr 또는 파일명
- [x] **CLI-02**: --emit-tokens — 토큰 목록 출력
- [x] **CLI-03**: --emit-ast — AST 출력
- [x] **CLI-04**: --emit-type — (예약) 타입 체킹 출력
- [x] **CLI-05**: emit 옵션을 활용한 파일 기반 테스트

## v2 Requirements (Future)

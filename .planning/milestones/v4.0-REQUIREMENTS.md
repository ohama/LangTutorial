# Requirements: LangTutorial v4.0

**Defined:** 2026-02-01
**Core Value:** 각 챕터가 독립적으로 동작하는 완전한 예제를 제공하여, 독자가 언어 구현의 각 단계를 직접 따라하고 실행해볼 수 있어야 한다.

## v4.0 Requirements

Requirements for v4.0 타입 시스템. Each maps to roadmap phases.

### Type Definition (타입 정의)

- [x] **TYPE-01**: Type AST 정의 (TInt, TBool, TString, TVar, TArrow, TTuple, TList)
- [x] **TYPE-02**: Scheme 타입 정의 (forall 변수 리스트 + 타입)
- [x] **TYPE-03**: TypeEnv 정의 (변수 이름 → Scheme 매핑)
- [x] **TYPE-04**: formatType 함수 (타입을 문자열로 변환)

### Substitution (대체 연산)

- [x] **SUBST-01**: apply 함수 (타입에 대체 적용)
- [x] **SUBST-02**: compose 함수 (대체 합성)
- [x] **SUBST-03**: freeVars 함수 (타입/스킴/환경의 자유 타입 변수)

### Unification (단일화)

- [x] **UNIFY-01**: occurs check (무한 타입 방지)
- [x] **UNIFY-02**: unify 함수 (두 타입을 같게 만드는 대체 찾기)
- [x] **UNIFY-03**: TypeError 예외 (타입 오류 메시지)

### Inference (타입 추론)

- [x] **INFER-01**: freshVar 함수 (새 타입 변수 생성)
- [x] **INFER-02**: instantiate 함수 (스킴을 타입으로 인스턴스화)
- [x] **INFER-03**: generalize 함수 (타입을 스킴으로 일반화)
- [x] **INFER-04**: infer 함수 — 리터럴 (Number, Bool, String)
- [x] **INFER-05**: infer 함수 — 산술/비교/논리 연산자
- [x] **INFER-06**: infer 함수 — 변수 참조 (Var)
- [x] **INFER-07**: infer 함수 — Let 바인딩 (let-polymorphism)
- [x] **INFER-08**: infer 함수 — Lambda와 App
- [x] **INFER-09**: infer 함수 — LetRec
- [x] **INFER-10**: infer 함수 — If 표현식
- [x] **INFER-11**: infer 함수 — Tuple
- [x] **INFER-12**: infer 함수 — List, EmptyList, Cons
- [x] **INFER-13**: infer 함수 — Match 표현식
- [x] **INFER-14**: infer 함수 — LetPat
- [x] **INFER-15**: inferPattern 함수 (패턴 타입 추론)

### Integration (통합)

- [x] **INTEG-01**: initialEnv 정의 (Prelude 함수 타입)
- [x] **INTEG-02**: typecheck 함수 (표현식 타입 검사)
- [x] **INTEG-03**: --emit-type CLI 옵션
- [x] **INTEG-04**: 타입 오류 시 프로그램 종료 (exit 1)

### Testing (테스트)

- [x] **TEST-01**: Type 모듈 단위 테스트
- [x] **TEST-02**: Subst 모듈 단위 테스트
- [x] **TEST-03**: Unify 모듈 단위 테스트
- [x] **TEST-04**: Infer 모듈 단위 테스트
- [x] **TEST-05**: TypeCheck 통합 테스트
- [x] **TEST-06**: fslit CLI 테스트 (--emit-type)
- [x] **TEST-07**: 타입 오류 케이스 테스트

## Out of Scope

| Feature | Reason |
|---------|--------|
| 타입 주석 구문 | 타입 추론만 지원, 명시적 주석 불필요 |
| 타입 클래스 | 복잡도 높음, v5.0+ 범위 |
| 레코드 타입 | ADT와 함께 v5.0+ 범위 |
| 재귀 타입 | 복잡도 높음, 기본 구현에 불필요 |
